# 🌍 CORC - Carbon Credit App for Small Farmers

**Democratizing carbon credits through satellite monitoring and mobile technology**

## 🎯 Mission

Help small farmers in developing countries access carbon credit markets through transparent satellite monitoring and instant micro-payments.

## 🚀 Project Status

- **Current Version:** v2.1 "GPS Revolution" 
- **Pilot Location:** Cape Verde
- **Next Milestone:** University partnership meeting (September 2025)

## 📱 Components

### Backend (FastAPI + Python)
- User authentication with JWT
- Google Earth Engine integration for NDVI calculations
- SQLite database
- RESTful API for mobile/web clients

### Mobile App (Flutter)
- GPS field mapping
- Photo capture with location data
- NDVI visualization charts
- Payment tracking interface
- Works on ANY Android device

### Features
- 🛰️ Satellite monitoring (Sentinel-2)
- 📍 GPS field boundary mapping
- 📊 NDVI vegetation health tracking
- 💰 Carbon credit calculations
- 📱 Mobile-first design

## 🔧 Tech Stack

- **Backend:** Python, FastAPI, SQLAlchemy, Google Earth Engine
- **Mobile:** Flutter, Dart
- **Database:** SQLite
- **Deployment:** Hetzner Cloud VPS
